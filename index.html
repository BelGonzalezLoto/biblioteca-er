📊 Diagrama Entidad-Relación - Trabajo Práctico N°2

Sistema de Biblioteca

Entidades: Objetos del mundo real
Relaciones: Asociaciones entre entidades
Cardinalidades: Restricciones numéricas
PK/FK: Claves primarias y foráneas

🧑‍🎓 ENTIDAD: AUTOR

Tipo: Entidad Fuerte

Clave Primaria: id_autor

Descripción: Representa a los escritores de los libros en la biblioteca.

Relación: Un autor puede ESCRIBIR múltiples libros (M:N con LIBRO)

📚 ENTIDAD: LIBRO

Tipo: Entidad Fuerte

Clave Primaria: ISBN

Descripción: Representa los libros disponibles en la biblioteca.

Relaciones: Puede ser escrito por múltiples autores (M:N) y puede ser incluido en préstamos (1:N)

👤 ENTIDAD: USUARIO

Tipo: Entidad Fuerte

Clave Primaria: id_usuario

Descripción: Representa a los usuarios de la biblioteca (estudiantes, profesores, personal).

Relación: Un usuario puede REALIZAR múltiples préstamos (1:N con PRÉSTAMO)

📋 ENTIDAD: PRÉSTAMO

Tipo: Entidad Fuerte

Clave Primaria: id_prestamo

Descripción: Representa las transacciones de préstamo de libros.

Claves Foráneas: Contiene referencias a USUARIO y LIBRO

✍️ RELACIÓN: ESCRIBE (M:N)

Entidades: AUTOR ↔ LIBRO

Cardinalidad: Muchos a Muchos

Notación Chen: ◊——————◊

Significado: Un autor puede escribir varios libros, y un libro puede tener varios autores.

Implementación: Requiere tabla intermedia con claves foráneas.

📝 RELACIÓN: REALIZA (1:N)

Entidades: USUARIO ↔ PRÉSTAMO

Cardinalidad: Uno a Muchos

Notación Chen: □——————◊

Significado: Un usuario puede realizar múltiples préstamos, pero cada préstamo pertenece a un solo usuario.

📖 RELACIÓN: INCLUYE (N:1)

Entidades: PRÉSTAMO ↔ LIBRO

Cardinalidad: Muchos a Uno

Notación Chen: ◊——————□

Significado: Múltiples préstamos pueden incluir el mismo libro (en diferentes momentos), pero cada préstamo incluye un solo libro.

📐 NOTACIONES DEL MODELO E-R

🔷 Notación Chen (Clásica):

  • Rectángulos: Entidades
  • Rombos: Relaciones
  • Óvalos: Atributos
  • Líneas: Conexiones
  • Números: Cardinalidades (1, N, M)

🦶 Notación Crow's Foot (Moderna):

  • || = Uno
  • }o = Muchos
  • o = Cero (opcional)
  • | = Uno exacto
  • Más visual y legible
💡 En este diagrama: Usamos la notación Chen con rectángulos para entidades, rombos para relaciones, y círculos numerados para cardinalidades.
